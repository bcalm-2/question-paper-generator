Computer Networks (CN) involve connecting computers to share resources. The OSI model and TCP/IP model are fundamental frameworks. Networking protocols like TCP, UDP, HTTP, and DNS enable communication across the internet. Routing is the process of selecting paths in a network. Network security ensures data privacy and integrity. DNS resolves domain names to IP addresses. HTTP and HTTPS are protocols for web communication. Socket programming is used for application-level networking. Wireless networks use radio waves instead of cables. OSI Model has 7 layers. TCP/IP has 4 layers. Routing algorithms find the best path.
