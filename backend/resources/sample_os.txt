An Operating System (OS) is software that manages computer hardware.
Processes are programs in execution.
Memory Management is the act of managing computer memory.
Deadlocks occur when two or more processes are waiting for each other to release resources.
Scheduling is the method by which work is assigned to resources.
First-Come, First-Served (FCFS) is the simplest scheduling algorithm.
Shortest Job Next (SJN) or Shortest Job First (SJF) is a policy that selects the process with the smallest execution time.
Round Robin (RR) scheduling is designed for time-sharing systems.
File Systems manage how data is stored and retrieved.
Virtual Memory gives an illusion of a large memory to the user.
Paging is a memory management scheme that eliminates the need for contiguous allocation.
Segmentation is a memory management technique that groups memory into logical segments.
Threads are the smallest unit of processing that can be performed in an OS.
Distributed Systems consist of multiple autonomous computers that communicate through a network.
Inter-process communication (IPC) refers to the mechanisms an OS provides to allow processes to manage shared data.
Mutexes and Semaphores are used for process synchronization.
A System Call is the programmatic way in which a computer program requests a service from the kernel.
The Kernel is the central component of an operating system.
Device Drivers allow the OS to communicate with hardware.
Bootstrapping is the process of starting the computer.
